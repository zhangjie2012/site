  在 WSL 系统中使用宿主 Windows 系统的代理，

  #+begin_src sh
	#!/bin/bash
	host_ip=$(cat /etc/resolv.conf |grep "nameserver" |cut -f 2 -d " ")
	export ALL_PROXY="http://$host_ip:1080"
  #+end_src

  如果访问不了，注意查看 Windows 系统防火墙。
