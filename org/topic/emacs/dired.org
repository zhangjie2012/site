我的使用习惯的 =<f9>-j= 绑定 =dired-jump= 快速跳转到当前文件所在的目录。为了子目录更好的操作，额外使用了 dired-subtree 插件。

你也可以使用 =C-x d= 进入 =dired= 。

* navigation

  | 快捷键 | 说明                          |
  | =q=    | 退出                          |
  | =g=    | 刷新                          |
  | =(=    | 显示、隐藏信息                |
  | =<=    | 移动到上一级目录，跳转到 =.=  |
  | =>=    | 移动到下一级目录，跳转到 =..= |
  | =^=    | 到上一级目录                  |
  | =s=    | 排序                          |
  | =i=    | 进入子目录                    |
  | =;=    | 关闭子目录                    |
  | =n=    | 下一个文件/目录               |
  | =p=    | 上一个文件/目录               |

* 选中

  | 快捷键 | 说明                   |
  | =m=    | 选中                   |
  | =u=    | 反选                   |
  | =U=    | 全部取消选中           |
  | =d=    | 标记为待删除           |
  | =t=    | 反选所有文件 mark 标记 |
  | =% m=  | 正则表达式选中         |
  | =% d=  | 正则表达是标记待删除   |

  =d= 选中的文件， =x= 开始执行。

* 文件/目录操作

  | 快捷键    | 说明                 |
  | =C-x C-f= | 新建文件             |
  | =+=       | 新建目录             |
  | =C=       | 拷贝文件             |
  | =D=       | 删除文件             |
  | =R=       | 重命名               |
  | =x=       | 删除被设置标记的文件 |
