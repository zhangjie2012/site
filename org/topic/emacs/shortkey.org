* Window + ace-windows

  |--------+-------------+----------------|
  | 前缀   | 后缀        | 说明           |
  |--------+-------------+----------------|
  | =C-x=  | =1=         | 只显示当前窗口 |
  |        | =2=         | 垂直分屏       |
  |        | =3=         | 水平分屏       |
  |        | =0=         | 当前窗口隐藏   |
  | =M-o=  | =1/2/3/...= | 切换窗口       |
  | =<f9>= | =w t=       | 交换窗口           |
  |--------+-------------+----------------|

* 多光标编辑（multiple-cursors）

  |-------+-------+------------|
  | 前缀  | 后缀  | 说明       |
  |-------+-------+------------|
  | =M-s= | =;=   | 当前函数   |
  |       | =>=   | 选中后一个 |
  |       | =<=   | 选中前一个 |
  |       | =C-a= | 全部选中   |
  |-------+-------+------------|

* 区域扩展（expand-region）

  |-------+------+------------------------------|
  | 前缀  | 后缀 | 说明                         |
  |-------+------+------------------------------|
  | =M=   | =m=  | expand-region                |
  | =M-s= | =s=  | 选中当前符号                 |
  |       | =p=  | 选中当前括号(包含括号本身)   |
  |       | =P=  | 选中当前括号(不包含括号本身) |
  |       | =q=  | 选中当前引用(包含 =`= )      |
  |       | =Q=  | 选中当前引用(不包含 =`= )    |
  |       | =m=  | 选中当前注释                 |
  |       | =f=  | 选中当前函数                 |
  |-------+------+------------------------------|

* 符号高亮（highlight-symbol）

  |------+------+------------------|
  | 前缀 | 后缀 | 说明             |
  |------+------+------------------|
  | =M=  | =-=  | 当前单词全部高亮 |
  |      | =n=  | 调转到下一个     |
  |      | =p=  | 跳转到上一个     |
  |------+------+------------------|

* avy

  |-------+------+------------------------------------|
  | 前缀  | 后缀 | 说明                               |
  |-------+------+------------------------------------|
  | =M-s= | =i=  | 根据单词的第一个字母定位，快速跳转 |
  |       | =j=  | 跳转到行                           |
  |       | =k=  | 拷贝指定行到当前                   |
  |-------+------+------------------------------------|

* ivy + counsel + swiper

  |----------+------+------------------------------|
  | 前缀     | 后缀 | 说明                         |
  |----------+------+------------------------------|
  | =C=      | =s=  | 当前文档搜索                 |
  | =M=      | =x=  | 覆盖默认 =M-x=               |
  |          | =y=  | 粘贴历史                     |
  | =M-s=    | =.=  | 当前文档搜索当前单词         |
  | =C-c=    | =g=  | 当前 git 中打开文件          |
  | =C-c=    | =k=  | 当前 git 项目中搜索, 使用 ag |
  | =<f9> g= | =r=  | 当前目录和子目录下搜索       |
  |          | =l=  | 只在当前目录搜索                     |
  |----------+------+------------------------------|

  一般用 =C-c p f= 代替 =C-c g= ，用 =C-c k= 代替 =<f9> s g= 。但是在 Windows 系统下无法使用 Projectile 和 ag。

* flycheck

  如果全局开启 flycheck，每次保存都会 check，这样会很卡，尤其是前端项目。所以关闭了自动检查：
  =(setq flycheck-check-syntax-automatically '())=

  然后绑定了手动检查的快捷键：

  |----------+-----------------|
  | =<f9> `= | 检查当前 buffer |
  | =<f9> l= | list errors     |
  |----------+-----------------|


* vc + git-gutter+

  |----------+-------+-----------------------|
  | 前缀     | 后缀  | 说明                  |
  |----------+-------+-----------------------|
  | =C-x=    | =n=   | 下一个修改            |
  |          | =p=   | 上一个修改            |
  |          | =r=   | revert 修改           |
  |          | ~v =~ | vc diff               |
  |          | =v l= | vc 日志               |
  | =<f9> g= | =g=   | enable/disable gutter |
  |----------+-------+-----------------------|

* org

  |-------------+--------------------------------------------------------------|
  | 快捷键      | 功能                                                         |
  |-------------+--------------------------------------------------------------|
  | =C-c C-e=   | Export or Publish                                            |
  | =C-c C-l=   | 插入链接等                                                   |
  | =C-c f=     | 插入脚注                                                     |
  | =C-c l=     | org-toggle-link-display                                      |
  | =C-c '=     | 在另外一个窗口编辑：代码，表格，公式等                       |
  | =C-c C-x p= | 插入标题 =PROPERTIES=                                        |
  | =C-c C-o=   | 打开当前链接 ~org-open-at-point~                             |
  | =C-c C-,=   | 插入结构化模板（类似代码，引用等），orgmode 9.2 版本之后支持 |
  | =C-c C-c=   | 标题上添加 Tag，可以用来分类或者打标签                       |
  |-------------+--------------------------------------------------------------|

  更多见 [[file:orgmode.org][org]]。

  | =<10>=     | 显示代码越界指示线                       |
  | =<f12>=    | counsel-imenu                            |
  |------------+------------------------------------------|
